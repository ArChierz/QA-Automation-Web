# QA Automation Web Portofolio

## To Run

### API Reqres
<b>Tech Stack</b>:
node fetch, ajv, mocha, chai, Mocha Simple HTML
```
npm install                     # initialization
npm run test                    # run API testing automation
npm run test-html               # run API Testing and report in HTML
npm run test-json               # run API Testing nd report in JSON
```
---

### Load Test
Use JMeter and load the HTTP Request.jmx, then click Run (green button).

---
### Web Saucedemo
<b>Tech Stack</b>:
Selenium, JS, Mocha, Mochawesome, Mocha Simple HTML, Chai
```
npm install                     # initialization
npm run test                    # run Web UI testing automation
npm run test-report-mochawesome     # run API Testing and report in HTML
npm run test-report-simple-html     # run API Testing nd report in JSON
```
